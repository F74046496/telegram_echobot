# Telegram Echobot
This branch is the minimal version of Telegram Echobot using Flask.

## Usage
1. Apply an Telegram Bot from BotFather
2. Change `'Your API Token'` and `'Your URL'` in `app.py` to proper value
3. `python app.py`
4. Start Chating with your bot
